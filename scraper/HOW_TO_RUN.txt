MATKA DATA SCRAPER - HOW TO RUN
================================

STEP 1: Install Dependencies
-----------------------------
pip install -r requirements.txt


STEP 2: Test the Scraper (Optional but Recommended)
---------------------------------------------------
python test_scraper.py

This will test scraping on one market to make sure everything works.


STEP 3: Run Full Scraper
-------------------------
python run_all.py

This will:
- Scrape the 12 trusted markets from dpbossss.boston
- Save data to data/ directory
- Analyze the data
- Generate reports


STEP 4: Check Results
----------------------
Look in the data/ folder:
- panel_data_TIMESTAMP.csv - All scraped data
- panel_data_TIMESTAMP.json - JSON format
- analysis_report_*.txt - Analysis for each market


TROUBLESHOOTING
---------------
If you get errors:

1. "No module named 'requests'"
   → Run: pip install -r requirements.txt

2. "No data scraped"
   → Check internet connection
   → Try: python test_scraper.py first

3. "Connection timeout"
   → Website might be slow
   → Try again later
